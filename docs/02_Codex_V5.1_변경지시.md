# Codex V5.1 변경 지시

`agentproof_v51_customer_discovery.html`과 같은 화면 및 문구로 기존 Next.js 구현을 수정한다.

우선순위:
1. V5.1 HTML과 PC·모바일 캡처
2. `01_V5.1_고객가설_콘텐츠_설계.md`
3. 기존 V5 기능·접근성·API 명세

필수 변경:
- 히어로와 모든 섹션 문구를 V5.1로 교체한다.
- 기존 “세 팀이 같은 화면에서 다른 결정을 내린다” 섹션과 5단계 공통 흐름을 삭제한다.
- 역할별 문제·서비스 카드 3개를 구현한다.
- 각 카드 CTA는 공통 진단 모달을 열고 역할을 자동 선택한다.
- 폼 필드를 역할, 단계, 문제, 후속 요청, 이메일, 선택 업무로 변경한다.
- `role_problem_click`, `diagnosis_start`, `diagnosis_submit`, 후속 요청별 분석 이벤트를 구현한다.
- 기존 대시보드 이미지와 AgentProof 브랜드, 모바일 대시보드, 접근성, UTM, 저장 API, 오류 처리는 유지한다.

검수:
- 1440px와 390px 전체 페이지 캡처를 생성한다.
- 320px 이상에서 가로 스크롤이 없어야 한다.
- 역할 카드별 자동 선택, 키보드 모달, 폼 성공·실패, 분석 이벤트를 E2E로 검증한다.
- V5.1 확정 카피를 임의로 늘리거나 한 카드에 두 문단 이상 추가하지 않는다.
