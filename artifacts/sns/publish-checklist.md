# SNS Publish Checklist

## Before Publishing

- [ ] Production URL opens over HTTPS.
- [ ] Role-specific URL keeps UTM parameters through survey start.
- [ ] URL contains no email, company, answer, or memo value.
- [ ] Role-specific Open Graph title and description are correct.
- [ ] Preview image renders clearly on mobile.
- [ ] Post copy does not claim certification, compliance, legal advice, or published customer results.
- [ ] Channel owner approves final copy.

## Manual Publishing Steps

1. Open the channel account in the authenticated browser or native app.
2. Paste the matching post copy from `artifacts/sns/post-copy.csv`.
3. Use the matching URL from `artifacts/sns/utm-links.csv`.
4. Confirm preview title, description, and image before publishing.
5. Publish only after account owner approval.
6. Record the real post URL and timestamp in this file.
7. Open the post URL in a clean browser and verify the survey link.
